# Catalog Icons

Local catalog icons used by the software catalog list.

The application stores these paths in `software_catalog.logo_url_large` and
`software_catalog.logo_url_small` instead of hotlinking registry or Gravatar
image URLs.

Sources:

- `apache-tomcat.png`: Apache Tomcat logo from `https://apache.org/logos/res/tomcat/`
- `redis.svg`: Simple Icons Redis SVG
- `nginx.svg`: Simple Icons NGINX SVG
- `mariadb.svg`: Simple Icons MariaDB SVG
- `prometheus.svg`: Simple Icons Prometheus SVG
- `apache-http-server.svg`: Simple Icons Apache SVG
- `nexus-repository.svg`: Simple Icons Sonatype SVG
- `grafana.svg`: Simple Icons Grafana SVG
- `elasticsearch.svg`: Dashboard Icons Elasticsearch SVG, based on Elastic branding

These files are used only for catalog item identification. Brand names and
logos may remain trademarks of their respective owners.
